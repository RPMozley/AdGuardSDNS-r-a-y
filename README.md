# AdGuardSDNS-r-a-y
Convert [r-a-y](https://github.com/r-a-y/mobile-hosts)'s AdGuard SDNS HOSTS format to Adblock format for use in AdGuard Home.

Using [r-a-y](https://github.com/r-a-y/mobile-hosts)'s converter.php to get latest [AdGuardDNS](https://adguardteam.github.io/AdGuardSDNSFilter/Filters/filter.txt) release & cleaned -> AdGuardDNS.txt

Automatically updated every 6 hours.
